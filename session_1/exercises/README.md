# Session 1 Exercises

This directory contains 3 progressive hands-on exercises for Session 1 (Practice Rounds 1-3).

## Exercise Overview

These exercises follow a progressive scaffolding model, reducing support as you build mastery:

### [Exercise 1: Workspace Configuration](exercise_1.md)
- **Duration:** 45 minutes
- **Type:** Hands-On Practice
- **Scaffolding:** High (step-by-step guidance)
- **Focus:** Setting up and configuring context-aware workspaces
- **Bloom Level:** Apply (Remember/Understand foundation)

### [Exercise 2: Workflow Acceleration](exercise_2.md)
- **Duration:** 36 minutes
- **Type:** Scenario Application
- **Scaffolding:** Medium (key decision points highlighted)
- **Focus:** Engineering context for debugging, feature development, and code review
- **Bloom Level:** Apply (with emerging Analyze skills)

### [Exercise 3: Advanced Diagnostics](exercise_3.md)
- **Duration:** 30 minutes
- **Type:** Guided Practice (Autonomous)
- **Scaffolding:** Low (minimal guidance)
- **Focus:** Diagnosing and fixing context issues across multiple workspaces
- **Bloom Level:** Analyze (diagnostic mastery)

## How to Use These Exercises

1. **Complete in order** - Each builds on previous skills
2. **Use available scaffolding** - Hints and examples are provided based on level
3. **Reflect thoroughly** - Reflection questions help consolidate learning
4. **Apply immediately** - Look for ways to use these skills in your real work
