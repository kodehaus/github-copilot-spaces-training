## 5-Point Context Audit Framework

Use this framework to audit each workspace:

1. **Workspace Files Complete?**
   - Are all necessary files included?
   - Is anything important missing?

2. **Documentation Accessible?**
   - Can the AI find and understand project documentation?
   - Is documentation clear and current?

3. **Errors/Logs Visible?**
   - Are error messages and logs available when needed?
   - Is debugging information accessible?

4. **Scope Appropriate?**
   - Is there too much or too little context?
   - Is the context focused on relevant information?

5. **Information Current?**
   - Is all information up-to-date?
   - Are there outdated examples or documentation?
