# Exercise 3 Resources: Workspace Audit Scenarios

This folder contains three real workspaces for you to audit. Each workspace has context issues that affect AI assistance quality.

## How to Use These Workspaces

1. **Examine each workspace folder** - Look at the files, structure, and content
2. **Apply the 5-Point Framework** - Evaluate each point systematically
3. **Identify specific issues** - Note concrete problems, not general observations
4. **Prioritize improvements** - Consider impact vs. effort
5. **Document your findings** - Use the deliverable template in exercise_3.md

## Evaluation Approach

For each workspace:

1. **Browse the files** - Spend 3-5 minutes exploring
2. **Note your observations** - What seems wrong or missing?
3. **Apply framework systematically** - Check each of the 5 points
4. **Test with a question** - Imagine asking the AI for help - what would go wrong?
5. **Prioritize fixes** - What would have the biggest impact?
