# TaskFlow Project

**Status:** Active Development  
**Team Size:** 8 developers  
**Stack:** React, Node.js, PostgreSQL

## Overview

TaskFlow is a project management application for agile teams. Started in 2020, currently at version 3.5.

## Features

- Task management
- Sprint planning  
- Team collaboration
- Real-time updates
- Reporting dashboard

## Technologies

- Frontend: React 18, TypeScript, Redux
- Backend: Node.js 20, Express, PostgreSQL 16
- Infrastructure: Docker, AWS
