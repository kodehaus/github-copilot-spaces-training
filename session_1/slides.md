# Core Features of GitHub Copilot Spaces

# Context Management Fundamentals

**Why Context Matters**

> "The quality of AI output is directly proportional to the quality of context provided."

**Good Context =** 
- Relevant files and documentation
- Clear instructions and constraints
- Appropriate scope and boundaries

**Poor Context =**
- Irrelevant or outdated information
- Vague or conflicting guidance
- Too broad or too narrow scope

---

# Context Strategy Impact

| Context Quality | AI Output Quality | Developer Experience |
|----------------|-------------------|---------------------|
| **Excellent** | Highly relevant, accurate | Fast, confident |
| **Good** | Mostly useful, some editing | Productive |
| **Poor** | Off-target, needs rework | Frustrating |
| **Missing** | Random, unusable | Wasted time |
